# Code of Conduct

OpenGovCon is committed to providing a respectful and professional environment.

## Expected Behavior

Participants are expected to:
- be respectful and constructive
- focus on technical and compliance issues
- avoid harassment, discrimination, or hostile conduct
- assume good faith in discussions

## Unacceptable Behavior

Unacceptable behavior includes:
- harassment or intimidation
- abusive language
- personal attacks
- disruptive behavior in technical discussions

## Enforcement

Project maintainers may:
- warn participants
- remove comments
- close issues
- ban repeat offenders

## Reporting

Report conduct issues to:

conduct@opengovcon.com
