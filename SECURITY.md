# Security Policy



## Supported Versions



OpenGovCon Reference Accounting System is an evolving reference implementation.



Security fixes will be applied to:

- the latest release branch (if releases exist)

- `main`



## Reporting a Vulnerability



Please do NOT report vulnerabilities publicly via GitHub issues.



Instead report them privately to:



security@opengovcon.com 



Include:

- affected version or commit hash

- reproduction steps

- impact description

- any suggested remediation



## Response Process



We will:

- acknowledge receipt

- evaluate severity and scope

- prepare a patch

- coordinate disclosure where appropriate



## Scope Notes



This system is designed for sensitive accounting and compliance data.



Security contributions should prioritize:

- authentication and authorization enforcement

- tenant isolation

- audit log integrity

- prevention of data deletion/mutation

- safe export/report generation
